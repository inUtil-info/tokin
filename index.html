This site demostrates an API walled garden by tokin

Tokin is a cloud service designed by inUtil Labs to verify who is making use of an API. This is a free demo and is not intended for commercial purposes.

To use this service, you will need to enter a valid token, which will be provided for free in the second step.

The walled garden API shows the whatsapp profile picture fetched on real time, obtained from a public URL that is not protected by login credentials.
